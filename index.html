Política de Privacidad

Esta aplicación utiliza la API de Facebook para permitir la publicación de contenido en páginas administradas por el usuario.

Información que recopilamos

La aplicación no almacena información personal sensible. Solo accede a los datos estrictamente necesarios proporcionados por Facebook, como el nombre del usuario y el identificador de la página.

Uso de la información

La información obtenida se utiliza únicamente para permitir la publicación de contenido en páginas de Facebook autorizadas por el usuario.

Almacenamiento y seguridad

Los tokens de acceso se almacenan de forma segura y no se comparten con terceros.

Eliminación de datos

El usuario puede solicitar la eliminación de sus datos escribiendo a: servicios.de.internet.col@gmail.com
